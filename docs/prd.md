# Project: Website Qurban, Aqiqah & Katering Jatiasih

## Overview
A modern, professionally-designed, and SEO-optimized website for a Qurban, Aqiqah, and catering business based in Jatiasih, Bekasi (Jl. Villa Jatirasa, Gg Rimban No 68). The goal is to maximize marketing reach and convert visitors into customers by showcasing high-quality services and routing all orders/inquiries directly to WhatsApp.

## Core Features
- [ ] **Dynamic Landing Page:** Highly visual hero section, clear value propositions, and testimonials/trust indicators.
- [ ] **Product/Service Catalog:** Detailed sections for Qurban animals, Aqiqah packages, and Event Catering with prices and descriptions.
- [ ] **WhatsApp Integration:** All "Beli" (Buy) or "Tanya" (Inquire) buttons deep-link directly to a pre-filled WhatsApp message.
- [ ] **SEO & Local Marketing Ready:** Fully configured meta tags, semantic HTML, modern structure, and Local Business schema to attract nearby customers.
- [ ] **Image Gallery & Content:** Rich placeholder and generated imagery to create a premium feel (no blank layouts).

## Tech Stack
- Frontend: Next.js (React) using App Router (for best SSG and SEO)
- Styling: Tailwind CSS configured with custom Design Tokens
- Backend & DB: None (Static data / JSON driven for pure frontend)
- Deployment: Vercel (recommended for Next.js)

## Out of Scope
- Online Payment Gateway / In-app checkout (All handled via WhatsApp)
- User Authentication / Login system
- Complex admin dashboard (CMS will be file-based/JSON for now)
- Automated inventory tracking

## UI/UX Direction (if frontend/mobile)
- **Color palette:** 
  - Primary: Emerald/Islamic Green (Trust, Freshness)
  - Secondary: Clean Off-White/Surface (Modern, spacious)
  - Accent: Warm Action Color (e.g., subtle Gold or vibrant WhatsApp Green)
- **Typography:** `Inter` or `Plus Jakarta Sans` for maximum readability and a premium modern look.
- **Layout type:** Landing + Catalog (highly visual, scannable).
- **Visual reference:** Premium halal marketplace, modern food catering, clean layout with generous whitespace.

## Success Criteria
- Website achieves 90+ Lighthouse score across SEO, Accessibility, and Performance.
- Intuitive user experience where visitors can find WhatsApp contact within 5 seconds.
- Premium visual aesthetic that distinguishes the business from competitors.
